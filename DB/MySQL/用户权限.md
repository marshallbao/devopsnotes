### 用户管理

```mysql
# 添加用户
> CREATE USER 'relayer'@'%' IDENTIFIED BY 'peit4scB';
# 删除用户并撤销其原有权限
> DROP USER 'relayer'@'%';

```



### 权限相关

```mysql
#权限查看
> show grants for relayer@%;

# 权限添加
> GRANT ALL PRIVILEGES  ON relayer.* TO 'relayer'@'%';
> GRANT ALERT,SELECT  ON relayer.* TO 'relayer'@'%';
> FLUSH PRIVILEGES;

# 权限回收
# 收回所有用户的所有权限
> REVOKE ALL PRIVILEGES FROM 'user'@'host' 

# 收回指定的权限
> REVOKE UPDATE ON db1.* FROM 'test2'@'localhost';
> FLUSH PRIVILEGES;
```
