# 备份恢复

pg_dump

```
# 备份数据
pg_dump -U username -d database_name > table_structure.sql

# 只导出数据结构
pg_dump -U username -d database_name -s -t table_name > table_structure.sql

# 指定表
pg_dump -U username -d database_name -s -t table_name > table_structure.sql



# 参数
-s 或 --schema-only 表示只导出表结构，不包括数据。
-t 或 --table 指定要导出结构的表名。
```
