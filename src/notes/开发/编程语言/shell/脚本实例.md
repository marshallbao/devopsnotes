脚本1：批量拉取 minio 数据

```shell
#!/bin/bash

<<INFO
DESCRIBE: pull minio objects
USE: bash wgetoss.sh dir
AUTHOR: yonggui
DATE: 20230117
WARNING: 
INFO

while read dd
do
    echo $dd
    mkdir -p /tmp/$1$dd
    for i in `mc ls osspro/$dd|awk '{print $5}'`
    do mc mirror osspro$dd$i /tmp/$1$dd/
    done
done<list

cd tmp && tar -czvf $1.tar.gz /tmp/$1
```
