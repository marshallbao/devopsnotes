### 角色

IAM 角色是可在账户中创建的一种具有特定权限的 IAM 身份

### 策略

策略是 AWS 中的 JSON 文档，可让您指定谁有权访问 AWS 资源，以及他们可以对这些资源执行哪些操作。您可以将策略附加到身份或资源以定义其权限





### 基于身份的策略



在什么条件下，对哪些资源进行哪些操作

### 基于资源的策略

是附件到资源的策略，授予指定委托人在特定的资源上执行特定的操作的权限

角色可绑定信任策略（基于资源的策略）和权限策略（基于身份的策略）
