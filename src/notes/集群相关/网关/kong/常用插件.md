# 常用插件

key-auth

```
# 为某个 service/route 启用插件
curl -X POST http://localhost:8001/apis/example-api/plugins/ \
--data "name=key-auth"

# 新建 consumer
curl -i -X POST \
--url http://192.168.150.41:31218/consumers/ \
--data "username=test-a"

# 为 consumer 配置 key
curl -i -X POST \
--url http://192.168.150.41:31218/consumers/test-a/key-auth/ \
--data 'key=12345678'

# consumer 可以用于不同的插件/认证方式
http://192.168.150.41:31218/consumers/test-a/hmac-auth/
http://192.168.150.41:31218/consumers/test-a/key-auth/
```
