# 配置管理

常见参数

```
# 以下副本数要少于 brokes 数量
# 创建 topic 的时候默认副本数
default.replication.factor

# 控制 Kafka 自动创建的内部 topic __consumer_offsets 的副本数
# __consumer_offsets 用于存储消费者组的 offset 信息（即消费进度），主要是使用 Kafka Consumer Group 
offsets.topic.replication.factor

# 控制 Kafka 自动创建的内部 topic __transaction_state 的副本数，这个 topic 用于支持 Kafka 的事务机制。
transaction.state.log.replication.factor

```
