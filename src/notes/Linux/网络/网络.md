### 日常操作

静态 ip 配置

```
# cd /etc/sysconfig/network-scripts/
# 找到对应的网卡
# vim  ifcfg-eth0
NAME="eth0"
ONBOOT="yes"
NETBOOT="yes"
BOOTPROTO="none"
TYPE="Ethernet"
IPADDR=10.2.0.20
GATEWAY=10.2.0.1
PREFIX=24
DNS1=202.96.209.133
DNS2=114.114.114.114
DEFROUTE="yes"

# 重启网络服务
# service network restart

```
