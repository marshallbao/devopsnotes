下载安装 golang

```
wget https://go.dev/dl/go1.19.3.linux-amd64.tar.gz
tar -xf go1.19.3.linux-amd64.tar.gz
```



查看环境变量

```
$ go env
//以JSON格式输出
$ go env -json
```

开启代理

```
# 开启 mod 管理包模式
$ go env -w GO111MODULE=on

$ go env -w GOPROXY=https://goproxy.cn,direct
```

关闭代理

```
#取消环境变量 GOPROXY
$ go env -u GOPROXY
$ go env -w GOPROXY=direct
```
