场景1：a 域名代理 b 域名，只允许 从 a 域名跳转到 b 域名的访问，不允许直接访问b 域名，使用 request_header 中的 referer 字段

```
    http {
      map $http_referer $allowed {
        default       0;
        "~^https?://avata.qa.nj.bianjie.ai"  1;
      }
    }

    location /api {
      if ($allowed = 0) {
        return 404;
      }
    }
```
